# IPA Typing Keyboard

A privacy-focused iOS keyboard for typing International Phonetic Alphabet
(IPA) symbols. Built for linguistics students, phoneticians, and language
learners.

## How it works

Install from the App Store, add **IPA Typing Keyboard** in
**Settings → General → Keyboard → Keyboards → Add New Keyboard**, then
switch to it with the 🌐 globe key in any app.

Long-press the dotted letters (a, e, i, o, u, t, s, d, c, n, z) to pick
an IPA variant — the same gesture you already know from the built-in
iOS é/è popover.

## Privacy

- **No Full Access required.** The keyboard runs in Apple's standard sandbox.
- **No network, no accounts, no analytics, no third-party SDKs.**
- **No typed text ever leaves your device.** We cannot read what you write.

See the [Privacy Policy](privacy.html) for full details.
